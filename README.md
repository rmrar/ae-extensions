# EllieTools

An After Effects CEP extension for motion designers. Built by rm.rar.

---

## Features

### Presets Tab
Import and manage After Effects animation presets (.ffx files). Click + Import .ffx to browse for a preset file, saved presets appear as buttons — select your layers then click to apply. Right-click a preset to remove it. Presets save between sessions.

### Tools Tab
Quick layer and comp utilities — freeze frame, fit to comp, adjustment layer, null, solid, camera, fill, blur, tint, center, and save PNG.

**Precomp** — click to precomp all selected layers together into one precomp. Alt+click to precomp each selected layer into its own separate precomp.

---

## Installation

1. Download and unzip the zip file from this repo
2. Copy the `EllieTools_v4_1` folder to your CEP extensions folder:
   - Windows: `C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\`
   - Mac: `/Library/Application Support/Adobe/CEP/extensions/`
3. Restart After Effects
4. Open via Window → Extensions → EllieTools

---

## Requirements
- After Effects CC 2019 or later
